# 00 Reglas Maestras

**Autor:** Roberto Monedero Alonso
**Proyecto:** "El Cuaderno de Campo Vivo" (App Ornitológica)

## Visión del Proyecto

Aplicación móvil híbrida de colección y estrategia en tiempo real. Combina la exploración física (Geolocalización tipo Pokémon GO para avistamiento y recolección de recursos) con batallas estratégicas de cartas (tipo Magic/Hearthstone) basadas en ornitología real.
El sistema incluye un Marketplace en tiempo real y funciones sociales avanzadas, soportado por una arquitectura backend Reactiva y No Bloqueante.
